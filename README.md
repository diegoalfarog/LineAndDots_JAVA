# Lines and dots
 An line and dots game crafted with heritage-polymorphism and matrixes in java

To run 
Make sure whether have installed  a JDK last versión 

Run in terminal 
    command : 




        //to Compile
        javac TestTimbiriche.java

        java TestTimbiriche.java
